Задание - решить дружной командой следующие задачи:
1 Создайте REST-контроллер, который анализирует все заголовки, начинающиеся с префикса «x-» (без учёта регистра).
Метод возвращает заголовки и их значения в виде списка пар в теле ответа. Если в запросе нет ни одного заголовка,
то метод возвращает ошибку с кодом 404 Not found. Если запрос содержит заголовки,
начинающиеся с русской буквы х (ха) вместо английской x (икс), то метод возвращает ошибку с кодом 400.
(используйте внедрение бинов HttpServletRequest и HttpServletResponse в метод контроллера.
Для этого не забывайте к прочим зависимостям добавить эту
https://mvnrepository.com/artifact/jakarta.servlet/jakarta.servlet-api)
